# Category Rules

Use these rules to keep category matching consistent.

## Restaurant

Include:

- restaurants
- cafes only if the user explicitly includes cafes or brunch
- bars only if the user explicitly includes drinks or nightlife

Prefer fields:

- cuisine or meal type
- price range
- reservation availability
- suitability tags such as `group`, `family`, `date`, `solo`

Exclude when:

- the venue is only a convenience store or grocery
- hours or category are too unclear to verify the request

## Parking

Include:

- public parking lots
- paid private parking lots
- on-site destination parking if clearly available to visitors

Prefer fields:

- distance to anchor
- public access status
- fee type
- operating hours
- EV charging if requested

Exclude when:

- parking is resident-only, staff-only, or private without public access confirmation
- parking is mentioned only indirectly with no supporting listing

## Surf Shop

Include:

- surf rental shops
- surf lesson operators with a physical shop or meeting point
- surf retail shops
- repair shops if the user asks for repair or gear service

Prefer fields:

- rentals
- lessons
- retail gear
- board repair
- shower or changing support

Exclude when:

- the business is only a general sports store with no surf evidence
- the business has no reliable location or contact evidence
