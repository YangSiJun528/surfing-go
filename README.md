# surfing-go

`surfing-go`는 국내 서핑 포인트 정보를 초보자도 이해하기 쉽게 풀어 보여주는 서비스입니다.

## 목표

- 서핑 입문자도 예보를 읽고 직접 포인트를 고를 수 있게 돕기
- 여러 해변의 컨디션을 한눈에 비교할 수 있게 만들기
- 파고, 파주기, 풍속, 수온 같은 정보를 쉬운 언어로 해석해 전달하기
- 현장 이동 전 필요한 핵심 판단을 빠르게 끝낼 수 있게 하기

## 핵심 기능

- 전국 주요 서핑 포인트를 지도 기반으로 확인
- 각 포인트의 현재 컨디션을 직관적인 등급으로 요약
- 초급, 중급, 상급 기준으로 서로 다른 추천 코멘트 제공
- 오늘 들어가기 좋은 시간대를 함께 안내
- 파고, 파주기, 풍속, 수온의 의미와 해석을 쉬운 문장으로 설명
- 각 해변 주변의 식사, 카페, 렌탈, 휴식 장소 등 로컬 정보 제공

## 이런 사용자에게 적합합니다

- 예보 앱 숫자는 보이지만 어떻게 해석해야 할지 어려운 초보 서퍼
- 여러 포인트 중 오늘 어디로 갈지 빠르게 정하고 싶은 사용자
- 단순한 날씨 확인이 아니라 실제 입수 판단까지 도움받고 싶은 서퍼

## 서비스가 지향하는 경험

기존 서핑 정보 서비스는 숙련자에게 익숙한 표현과 수치 중심으로 구성된 경우가 많습니다. `surfing-go`는 그 장벽을 낮추는 데 초점을 맞춥니다. 전문 예보를 초보자 관점의 언어로 다시 정리해, "보기는 했지만 이해는 못한 정보"를 "바로 행동으로 옮길 수 있는 정보"로 바꾸는 것이 이 프로젝트의 방향입니다.
