# 서프레디

`서프레디`는 초보자도 이해하기 쉬운 국내 서핑 포인트 안내 서비스입니다.

## 소개

`서프레디`는 국내 주요 서핑 포인트를 초보자 기준에서 더 쉽게 이해하고 비교할 수 있도록 돕는 서비스입니다.

## 핵심 기능

- 전국 주요 서핑 포인트를 지도 기반으로 확인
- 각 포인트의 현재 컨디션을 직관적인 등급으로 요약 (해양수산부 국립해양조사원 서핑지수 조회 API)
- 파도 높이, 바람 등 서핑에 영향을 미치는 요소들을 초보자 기준으로 쉽게 설명
- 포인트별 추천 코멘트와 포인트 주변 정보까지 함께 제공

## 개발 배경

기존 파도 예보 서비스는 초보자에게는 숫자와 용어가 너무 많고 복잡하게 느껴질 수 있습니다.
숙련자에게는 익숙한 표현과 수치들이겠지만, 초보자는 예보를 봐도 바로 "오늘 어디로 몇 시에 가야 하는지", "그렇다면 왜 그런지, 예보만 믿어도 되는지" 판단하기 어렵습니다.

저희 팀은 초심자를 위한 파도 확인 서비스를 통해, 그들이 스스로 파도를 예측하고 읽는 것이 익숙해지기 전까지 서핑을 포기하지 않도록 편의성을 제공합니다. 

## 목표와 지향점

- 초보자도 파도 예보 및 실시간 파도 상태를 읽고 직접 포인트를 고를 수 있게 하기
- 복잡한 수치를 바로 행동 가능한 정보로 바꾸기
- 서핑 입문자가 부담 없이 서핑 계획을 세울 수 있게 돕기
